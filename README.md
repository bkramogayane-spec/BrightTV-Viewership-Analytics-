# BrightTV Case Study – User & Usage Analytics
# Description
This repository contains analysis and recommendations for BrightTV to grow its subscription base and improve content consumption. It includes:
1. Data cleaning and preparation
2. Exploratory analysis of user profiles and viewership trends
3. Insights on factors influencing consumption
4. Content and strategic recommendations
5. Dashboard layout for monitoring KPIs
 
# Project Objective
1. Analyze BrightTV User and Usage Trends
Provide insights into how users engage with BrightTV (e.g., active users, viewing frequency, peak usage times, and content preferences).
2. Identify Key Factors Influencing Consumption
Explain what drives or limits viewing behavior (e.g., time of day, device type, content genre, pricing, promotions, external events).
3. Recommend Content Strategies for Low-Consumption Days
Suggest specific content types or formats that can boost engagement during periods of low activity (e.g., short-form videos, trending genres, live events).
4. Propose Initiatives to Grow BrightTV’s User Base
Outline actionable strategies to attract new users and retain existing ones (e.g., partnerships, personalized recommendations, loyalty programs, marketing campaigns).

# Data Sources
BrightTV_UserProfiles.csv – User demographics and profile info

BrightTV_Viewership.csv – Session-level viewing data (channel, date, duration)

# Analysis Workflow
1. Data Cleaning: Handle missing values, normalize date/time to SA time zone
2. Exploratory Analysis: User demographics (age, province) and Usage patterns (sessions per day, duration)
3. Visualization : Peak viewing hours, Channel popularity and Consumption trends by day/week
4. Recommendations: Content strategy for low-consumption days and Initiatives for user growth

# Findings
# ✅ 1. User & Usage Trends
BrightTV offers 19 channels ranging from kids to adult entertainment.
Current subscription base: ~10,000 users.
# Top-performing channels:
SuperSport Live Events (17.94%) and ICC Cricket World Cup 2011 (15.82%) dominate sessions → strong demand for live sports.
kykNET (~1450 sessions) shows high engagement for local entertainment.
Channel O (11.34%) and Trace TV (10.28%) popular for music/youth content.
# Low-engagement channels: 
MK, M-Net, Wimbledon, Break in Transmission → minimal sessions.

# ✅ 2. Viewing Behavior
# Average viewing time trends:
March shows the highest engagement across channels.
ICC Cricket World Cup 2011 peaks at ~17 minutes in March → event-driven loyalty.
Trace TV & Channel O maintain ~10–12 minutes consistently → strong music/youth appeal.
Cartoon Network & Africa Magic moderate at 6–8 minutes.

# ✅ 3. Demographics
# By Province:
Gauteng leads (urban, high connectivity), followed by Western Cape.
KwaZulu-Natal & Mpumalanga moderate; rural provinces lag due to low broadband penetration.
# By Age & Gender:
26–35 age group dominates, followed by 36–45 → tech-savvy, stable income, strong digital adoption.
18–25 moderate (prefer short-form/social media).
46+ drops sharply → slower streaming adoption.

Overall, Male users lead across all age groups, likely due to sports/news preference.

# ✅ 4. Factors Influencing Consumption
Urbanization & connectivity drive viewership.

Content type: Sports and music channels outperform others.

Demographics: Younger and middle-aged users consume more digital content.

Event-driven spikes: Sports tournaments boost engagement.

# ✅ 5. Content Recommendations for Low-Consumption Days
Promote top channels (sports, music) during off-peak days.
Refresh content on low-performing channels.
Shorter ads and cross-promotion to maintain engagement.

# ✅ 6. Initiatives to Grow User Base
Personalization: Tailor content for 26–45 age group; experiment with youth-focused formats.

Retention: Loyalty rewards for frequent viewers; improve user profiling.

Regional reach: Community-driven campaigns; affordable data bundles for rural areas.

Acquisition: Enhance onboarding, mobile streaming experience, and offer free trials.

# ✅ Strategic Goal
Increase subscription base by leveraging sports & music content, personalized experiences, and regional expansion strategies.
